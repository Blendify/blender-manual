
***********
Dynamic Sky
***********

Introduction
============

The documentation can be found on the `author's site <http://www.dragoneex.com/downloads/dynamic-skyadd-on>`__.


Activation
==========

- Open Blender and go to Preferences then the Add-ons tab.
- Click Lighting then Dynamic Sky to enable the script.
