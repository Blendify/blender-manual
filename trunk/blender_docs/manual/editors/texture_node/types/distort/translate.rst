
**************
Translate Node
**************

.. figure:: /images/render_blender-render_textures_nodes_types_distort_translate_node.png
   :align: right

   Translate node.

Translate the texture coordinates of an image or texture.


Inputs
======

Color
   Standard image input.
Offset
   The amount to offset the coordinates in each of the three axes.


Properties
==========

This node has no properties.


Outputs
=======

Color
   Standard image output.
