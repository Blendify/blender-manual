***********
Normal Node
***********

.. figure:: /images/compositing_nodes_normal.png
   :align: right
   :width: 150px

   Normal Node.

The Normal node generates a normal vector and a dot product.

Input
=====

Normal
   Normal vector input.


Properties
==========

Normal Direction
   :kbd:`RMB` click and drag on the sphere to set the direction of the normal.

Output
======

Normal
   Normal vector output.
Dot
   Dot product output.
