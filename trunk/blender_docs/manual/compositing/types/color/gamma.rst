**********
Gamma Node
**********

.. figure:: /images/compositing_nodes_color_gamma.png
   :align: right
   :width: 150px

   Gamma Node.

Use this node to apply a gamma correction.


Inputs
======

Image
   Standard image input.
Gamma
   An exponential brightness factor.


Properties
==========

This node has no properties.


Outputs
=======

Image
   Standard image output.
