**************
RGB to BW Node
**************

.. figure:: /images/compositing_nodes_converter_rgb-to-bw.png
   :align: right

   RGB to BW Node.

The *RGB to BW Node* maps a RGB color image to a grayscale by the luminance.


Inputs
======

Image
   Color image input.


Properties
==========

This node has no properties.


Outputs
=======

Value
   Grayscale value output.
