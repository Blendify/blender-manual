
******************
Subdivide Modifier
******************

The *Subdivide* modifier subdivide the strokes adding points to the lines.


Options
=======

.. figure:: /images/grease_pencil_modifiers_generate_subdivide_panel.png
   :align: right

   The Subdivide modifier.

Level
   The number of subdivision iterations.

   Simple
     When enabled, the subdivide proccess only add control points.

Influence filters
-----------------

Material
   Restricts the effect only to material that share the same pass index.

Layer
   Restricts the effect only to one layer or to any layers that share the same pass index.
