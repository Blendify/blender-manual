
************
Introduction
************

*Grease Pencil* is a particular type of Blender Object that allow you to draw in the 3D space. 
Can be use to make traditional 2D animation, cut-out animation, motion graphics or use it as storyboard tool among other things.


.. figure:: /images/grease_pencil_introduction_example.png

   An example of a Grease Pencil Object in the 3D environment.

The Object act as a container of strokes that you can create using drawing tools in :doc:`Draw Mode </grease_pencil/modes/draw/introduction>`.
More detailed editing of the strokes is done in :doc:`Edit Mode </grease_pencil/modes/edit/introduction>`, and :doc:`Sculpt Mode </grease_pencil/modes/sculpting/introduction>`.
