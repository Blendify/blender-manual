
************
Checker Node
************

.. figure:: /images/render_blender-render_textures_nodes_types_patterns_checker_node.png
   :align: right

   Checker node.

The checker node creates a checkerboard pattern.


Inputs
======

color 1, color 2
   Image inputs setting the color of the squares.
Size
   The scale of the checker pattern.


Properties
==========

This node has no properties.


Outputs
=======

Color
   Standard image output.
